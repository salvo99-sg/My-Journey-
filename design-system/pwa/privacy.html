Privacy Policy

Last updated: July 2026

Overview

My Journey is designed following a Local First philosophy. Personal travel information is stored locally on the user's device unless future cloud synchronization is explicitly enabled.

Collected Data

  • Journey information
  • Travel notes
  • Budget data
  • Gallery metadata
  • Application preferences

Storage

Application data is primarily stored locally using browser storage technologies.

User Control

Users may export, import or permanently delete their data at any time.

Contact

Privacy requests should be directed to the official project maintainers.
