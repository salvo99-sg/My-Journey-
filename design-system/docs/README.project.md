# My Journey

Premium Travel Planner & Digital Travel Journal

---

## Overview

My Journey is a premium Progressive Web App designed to organize, experience and preserve every journey.

The application follows a **Local First** architecture and is entirely built with:

- HTML5
- CSS3
- Vanilla JavaScript
- Progressive Web App technologies

No frontend framework is required.

---

## Features

- Journey Planner
- Interactive Timeline
- Offline Maps
- Budget Management
- Travel Journal
- Gallery
- Tickets
- Packing Lists
- Weather
- Hotel Management
- Memories
- Offline Mode
- Dark Mode
- Multi-language
- PWA Installation
- Automatic Local Backup
