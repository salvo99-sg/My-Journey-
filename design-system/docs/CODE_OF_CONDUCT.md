# CODE_OF_CONDUCT

# My Journey

## Our Standards

Contributors are expected to:

- Be respectful.
- Communicate professionally.
- Provide constructive feedback.
- Focus discussions on improving the project.
- Respect project decisions and coding standards.

---

## Unacceptable Behavior

The following behaviors are not accepted:

- Harassment
- Personal attacks
- Discrimination
- Offensive language
- Deliberate disruption of development
- Malicious code contributions

---

## Development Expectations

Every contribution should:

- Follow the documented architecture.
- Respect the Design System.
- Preserve accessibility.
- Maintain code quality.
- Include appropriate documentation when required.

---

## Enforcement

Project maintainers reserve the right to reject or remove contributions that do not comply with these guidelines.

The primary objective is to maintain a professional, collaborative and high-quality development environment.
