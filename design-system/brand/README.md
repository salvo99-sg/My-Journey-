# Brand assets — My Journey

Asset del marchio definitivo. I **primari** (`logo-primary`, `logo-symbol`, `logo-lockup`)
sono ricavati dal PNG hi-res dedicato (1536×1024). Le **varianti** sono ritagliate dalla
tavola ufficiale `logo-variants-sheet.png` (versione alta qualità).

| File | Cos'è | Uso |
|---|---|---|
| `logo-primary.png` | Lockup completo su bianco (sorgente hi-res) | archivio / stampa |
| `logo-lockup.png` | Lockup completo, sfondo trasparente | splash (light) |
| `logo-symbol.png` | Solo emblema (libro+monti+sole+aereo+penna), trasparente | app icon / favicon |
| `logo-wordmark.png` | Solo "My Journey" + tagline + B&C, trasparente | header / footer |
| `logo-mono-dark.png` | Monocromatica navy | fondi chiari |
| `logo-mono-light.png` | Monocromatica taupe chiara | fondi scuri / dark mode |
| `logo-gold.png` | Variante oro | usi premium / accenti |
| `logo-transparent.png` · `logo-full.png` | lockup trasparente / rifilato su bianco | utility |
| `logo-variants-sheet.png` | Tavola ufficiale 12 varianti | riferimento |

Icone app/PWA e favicon generate dall'emblema: vedi `../pwa/icons/`.

**Pendente dal designer:** versione **SVG vettoriale** del simbolo (per `mask-icon.svg`
Safari e per scalabilità infinita). Tutti gli asset qui sono raster.
